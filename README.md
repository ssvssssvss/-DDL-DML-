**Задание 1. Кеширование**
Приведите примеры проблем, которые может решить кеширование.
Приведите ответ в свободной форме.

**Ответ**
Кеширование помогает решать проблемы, связанные с производительностью и нагрузкой на системы:
1. Проблемы с повторяющимися вычислениями: тяжелые вычисления (генерация статистики, аналитики) можно кешировать, чтобы не повторять их каждый раз.
2. Ограничения внешних API – если приложение использует сторонние сервисы с лимитом запросов, кеширование помогает хранить результаты и уменьшить количество обращений к этим сервисам.
3. Задержки при формировании страниц или отчётов – сложные отчёты или динамические страницы, которые требуют объединения большого объёма данных, можно кешировать, чтобы пользователи получали готовый результат мгновенно.
4. Высокая нагрузка на серверы – при большом количестве пользователей одни и те же данные запрашиваются многократно. Кеш позволяет снизить количество вычислений и обращений к базовым сервисам.
5. Замедленные запросы к базе данных – если приложение часто делает одинаковые запросы к СУБД, кеширование позволяет хранить результаты этих запросов в памяти (Redis или Memcached) и отдавать их мгновенно, без обращения к диску.
